01

Reservas Online y Gestión de Cupos

El objetivo principal era digitalizar la captación de clientes. Implementé un sistema de reservas online donde el usuario selecciona su fecha de salida y, si hay plazas ("30 disponibles"), puede rellenar el formulario de reserva al instante.

A nivel interno, el administrador mantiene el control total, pudiendo abrir o cerrar fechas manualmente según la estrategia comercial, sin depender de automatismos rígidos.

  • Formulario de reserva integrado.
  • Visualización de plazas libres.
  • Solicitud directa desde la ficha del viaje.
02

Área Privada B2B para Agencias

Implementación de una arquitectura de Usuarios y Roles. Mediante un sistema de Login seguro, las agencias colaboradoras acceden a un panel exclusivo.

Al autenticarse, la web adapta la información mostrada, permitiendo a los colaboradores acceder a documentación interna y visualizar las condiciones comerciales específicas para profesionales.

  • Autenticación segura (Login/Registro).
  • Panel exclusivo para colaboradores.
  • Acceso a documentación B2B.
03

Visualización de Itinerarios Completos

Estructuración de datos para mostrar productos turísticos detallados. Cada viaje se desglosa en un itinerario día a día fácil de consumir.

Integración de folletos, rutas y galerías multimedia organizadas por pestañas, ofreciendo toda la información técnica necesaria para el peregrino de forma limpia y ordenada.

  • Desglose de contenido por días.
  • Integración de folletos.
  • Contenido multimedia.
