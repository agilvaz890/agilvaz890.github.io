01

Motor de Búsqueda y Filtrado

Para mejorar la Experiencia de Usuario (UX), implementé un motor de búsqueda optimizado. Al escribir términos como "gargantilla" o "pendientes", el usuario puede encontrar productos relacionados rápidamente.

Esto se combina con filtros laterales de rango de precios y categorías para segmentar el catálogo.

  • Resultados relacionados al instante.
  • Filtrado dinámico de productos.
  • Reducción de tasa de rebote.
02

Estrategia de Retención (Wishlist)

Integré una Lista de Deseos que permite guardar productos favoritos haciendo clic en el corazón.

Además, se permite mover productos directamente de la Wishlist al Carrito, facilitando el proceso de compra al máximo.

  • Guardado persistente (Cookies/Sesión).
  • Traspaso directo Wishlist -> Carrito.
  • Gestión de stock visual.
03

Pagos Seguros: Redsys y Bizum

Integración técnica compleja de pasarelas de pago. Configuré el TPV Virtual oficial de Redsys para tarjetas bancarias y el módulo de Bizum, esenciales para el mercado español.

El checkout valida los datos en tiempo real y garantiza transacciones encriptadas bajo normativa PSD2.

  • Encriptación SSL en pasarela bancaria.
  • Integración nativa de Bizum.
  • Integración de pagos con PayPal.
  • Validación de dirección y envío.
